# Topcrete
E-Commerce Project built with MERN Stack

## Features
- Admin Dashboard
- Admin can Create, Update, Delete a product
- Admin User Management
- Database Seed script

## Usage
### Installing dependencies
```
npm i
cd frontend
npm i
```

### Run
```
# Run both Front-end & Back-end
npm run dev
# Run Backend only
npm run server
```

### Seed Database
```
// seed data
npm run data:import
// destroy data
npm run data:destroy
```
